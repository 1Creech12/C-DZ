# Улучшение расписания: Week View и Month View

## Описание изменений

Переработана система отображения расписания для режимов "Неделя" и "Месяц" с красивым оформлением карточек уроков.

## Что было сделано

### 1. Week View - Сетка дней недели ✓
- Сетка из 7 кнопок дней (2 строки: 4 кнопки + 3 кнопки)
- Каждая кнопка показывает день недели (ПН, ВТ, СР и т.д.) и дату
- Выбранный день подсвечивается зелёным цветом (#4CAF50)
- При клике на кнопку день загружается расписание для этого дня
- Навигация "Назад"/"Вперёд" переключает недели

### 2. Month View - Календарь с выбором дня ✓
- Встроенный QCalendarWidget для красивого отображения
- При клике на день в календаре загружается расписание
- Стилизация в Material Design стиле (зелёный цвет #4CAF50)
- Навигация "Назад"/"Вперёд" переключает месяцы

### 3. День View - Без изменений ✓
- Контейнер дней скрыт в DayView
- Календарь также скрыт
- Отображается только расписание текущего дня

### 4. Красивое оформление расписания ✓
Каждый урок отображается как карточка с:
```
┌─ Математика (жирный текст)
├─ 09:00 - 10:30 (серый текст)
├─ Аудитория: 301 (серый подтекст)
└─ Преподаватель: Иванов И.И. (серый подтекст)
```

- Левая цветная полоса (зелёный #4CAF50 для лекций, синий #2196F3 для практик)
- HTML-форматирование для красивого вывода
- Фон карточки: #f9f9f9
- Прозрачный фон QListWidget (#fafafa)

## Технические детали

### Новые члены класса ScheduleWidget
```cpp
QWidget *m_daysContainer;              // Контейнер для дней (неделя/месяц)
QVector<QPushButton*> m_dayButtons;    // Кнопки дней для WeekView
QCalendarWidget *m_calendarWidget;     // Календарь для MonthView
QPushButton *m_currentSelectedButton;  // Текущая выбранная кнопка дня
```

### Новые методы
- `createWeekDayButtons(QDate)` - создание сетки кнопок для недели
- `updateDayButtonStyles()` - обновление стилей кнопок (выбран/не выбран)
- `onDayButtonClicked()` - слот обработки клика на кнопку дня
- `onCalendarDayClicked(QDate)` - слот обработки клика в календаре

### Переработанные методы
- `updateWeekView()` - теперь создаёт кнопки дней вместо загрузки всей недели
- `updateMonthView()` - теперь показывает календарь вместо всех уроков месяца
- `displayLessons()` - теперь использует HTML для красивого форматирования
- `onViewModeChanged()` - добавлено переключение видимости контейнеров
- `onDayScheduleLoaded()` - работает для всех видов (не только DayView)

## Как работает

### WeekView режим:
1. Пользователь выбирает "Неделя"
2. Показывается сетка из 7 кнопок дней текущей недели
3. Первый день недели выбран по умолчанию (подсвечен зелёным)
4. При клике на кнопку дня:
   - Кнопка подсвечивается зелёным
   - Загружается расписание для этого дня
   - Расписание показывается в списке ниже
5. Кнопки "Назад"/"Вперёд" переключают недели

### MonthView режим:
1. Пользователь выбирает "Месяц"
2. Показывается календарь на текущий месяц
3. При клике на день в календаре:
   - День выбирается в календаре
   - Загружается расписание для этого дня
   - Расписание показывается в списке ниже
4. Кнопки "Назад"/"Вперёд" переключают месяцы

### DayView режим:
1. Пользователь выбирает "День"
2. Контейнер дней и календарь скрываются
3. Показывается расписание текущего дня
4. Кнопки "Назад"/"Вперёд" переключают дни

## Асинхронность
- Все операции загрузки расписания асинхронны через `ScheduleManager`
- При клике на день сначала отправляется сигнал загрузки
- При получении сигнала `dayScheduleLoaded()` отображается расписание
- Возможно отображение кэшированных данных пока идёт синхронизация с сервером

## Проверка перед сборкой
1. Убедитесь что Qt 6.11+ установлен
2. Выполните: `cmake .. -DCMAKE_BUILD_TYPE=Debug && make`
3. Запустите приложение и протестируйте все режимы:
   - Переключение между День/Неделя/Месяц
   - Клик на дни в каждом режиме
   - Кнопки Назад/Вперёд/Сегодня
   - Отображение расписания (карточки уроков)
   - Синхронизация при наличии интернета
